搭建图床

发表于2023-05-18|更新于2023-05-18|教程
|阅读量:

图床就是云端相册，可以解决写文章时插入图片的麻烦

创建图片仓库

这里利用Github

  • 创建仓库
  • 获取token
    • settings -> developer settings -> personal access tokens -> tokens(classic) -> generate new token(classic)
    • note随便设置
    • expiration设为永久no expiration
    • 勾选repo
    • 生成token

上传图片到仓库

这里利用PicGo

  • 下载PicGo
    • 我下载的是2.3.1 Latest
  • PicGo设置
    • 显示的图床只勾选Github
  • 图床设置
    • 仓库名为用户名/仓库名
    • 存储路径随意，但是最后一定要有个/
    • 自定义域名为cdn加速，可选

加速图片访问

这里利用staticaly

  • PicGo里面Github设置自定义域名设定为https://cdn.staticaly.com/gh/wannanap/ImgHub@master

至此，图床搭建完成！

使用

  • 打开PicGo
  • 选择上传区
  • 选择为github
  • 拖动图片上传即可
  • 复制时去相册
  • 选择复制格式进行复制即可

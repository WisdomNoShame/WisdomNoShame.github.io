hexo博客插入图片

发表于2023-05-13|更新于2023-05-13
|阅读量:

  • 刚好碰到写博客要插入图片的情况，普通的md文件直接插入，但不知道hexo写博客怎么插入图片

  • 参考文章：hexo博客如何插入图片 - 知乎 (zhihu.com)

图片插件

  • 有三个方法，但是用图片插件最有用、最方便

  • hexo-rendered-marked插件

    • 命令npm install hexo-renderer-marked直接安装

    • 安装后去_config.yml改配置

    • 现在就可以用同目录下的![](image.jpg)来愉快地插入图片啦！
